# ⏱️ JavaScript Stopwatch

A clean, minimalist, and highly accurate web-based stopwatch application built as part of my frontend development training and internship preparation. 

---

## 🎯 Project Objective
The purpose of this project was to practice working with JavaScript asynchronous functions, state management (start, pause, reset), and DOM manipulation to create a seamless, real-time user interface.

---

## ✨ Key Features
* ⏱️ **Precise Time Tracking** – Accurately counts minutes, seconds, and milliseconds.
* 🔄 **State Controls** – Fully functional Start, Pause, and Reset buttons that update dynamically.
* 📱 **Responsive UI** – A clean, centered, and modern design that looks great on both mobile and desktop screens.
* ⚡ **Optimized Logic** – Uses efficient JavaScript intervals to ensure smooth performance without lagging.

---

## 🛠️ Built With
* **HTML5** – For the core layout structure and control buttons.
* **CSS3** – For modern styling, flexbox centering, and button hover animations.
* **JavaScript (Vanilla)** – For handling the core timing logic using `setInterval`.
